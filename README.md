# FiveBadIdeas
Miniproject for Cyber Security Base Course. Tecs: Java - Maven - Springboot - Thymeleaf

## Usage from command line

### Testing

```
mvn test
```

### Running the application

```
mvn spring-boot:run
```

### Dependency check

```
mvn dependency-check:check
```

### Jar generation

```
mvn package
```
